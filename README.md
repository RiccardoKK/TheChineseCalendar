如何使用?
打开memorial-days.ics后
点击右上角的raw
在浏览器地址栏中获取到
https://raw.githubusercontent.com/RiccardoKK/TheChineseCalendar/refs/heads/main/memorial-days.ics
之后打开日历 找到新建日历订阅 输入刚刚在地址栏里复制到的raw地址完成订阅
